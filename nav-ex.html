*This code would typically be put inside of the first row in your grid.

The class="active" is added to the opening <a> tag of whatever page the user is on - notice in the example below, that the link for Nav Bar Example has the class="active" on its opening tag.

You must add class="active" to the appropriate link on the nav bar in each of your HTML documents.

					
						<ul class="nav nav-tabs">
								<li role="presentation"><a href="#">Grid Example</a></li>
								<li role="presentation" class="active"><a href="#">Nav Bar Example</a></li>
								<li role="presentation"><a href="#">Images with Bootstrap</a></li>
								<li role="presentation"><a href="#">Aligning Grid</a></li>
						</ul>

Customizing nav bar appearance:

Take a look at this link to learn what things like :hover and :focus mean: Mozilla Developer Network

As always when using Bootstrap, you can override any styles you so choose. In this example, we can remove the border added by Bootstrap.

The following styles are added to the CSS to remove all borders:

					
						.nav, .nav-tabs > li.active > a, .nav-tabs > li.active > a:hover,
						.nav-tabs > li.active > a:focus, .nav-tabs > li > a {
							border: none;
						}

The following styles are added to the CSS to remove the tab borders:

					
						.nav-tabs > li.active > a, .nav-tabs > li.active > a:hover,
						.nav-tabs > li.active > a:focus {
							border: none;
						}
