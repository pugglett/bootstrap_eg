Tips on alignment

Warning: Using the CSS detailed below may cause issues in how your page becomes responsive. In addition, these styles may not work in every web browser. They are optional to add to your page.

To add space inside of your grid, you can use empty <div> tags with columns added to them.

Example: <div class="col-md-2"></div>

All you need is some simple HTML, and a few lines of CSS added to your style.css document.

Add class="row-eq-height" to each of the rows in your grid that you want to have the same height.

Add the following CSS to your style.css document:

					
						.row-eq-height {
							display: -webkit-box;
							display: -webkit-flex;
							display: -ms-flexbox;
							display: flex;
						}

In order to add a background color to the empty div tags, a class is created.

This website uses the classes filler-pink, filler-green, filler-blue, and filler-grey to apply background colors to 'filler' columns.

				
						.filler-pink {
							background-color: #FFEDDF;
						}

Create whatever filler classes you would like.
