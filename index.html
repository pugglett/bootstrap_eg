This page covers examples of grid layouts.

  • Text is centered by adding class="text-center" to the opening tag of a text element like a <p> tag or <h1> tag.
  • Classes can be combined on an opening tag by putting a space in between them.

Remember the following when you are creating your grid system:

  • Everything on your page goes inside of a container (in this case, a <div class="container">)
  • Immediately inside of the container, are rows. There can be as many rows as you like (on this page, there are 11.) Rows are typically written using semantic layout elements like header, footer, section, and so on. You add the row class like this: <header class="row">
  • Immediately inside of each row are columns. Columns can be <div> tags.
  • Each division of a row must add up to but not over 12 columns - there should only be 1 element in each row for each division. For example, a row with 3 divisions of 4 columns each would have 3 elements inside of it, each with the appropriate 4 column class on it.
  • Determine the number of columns an element will take up by adding the col-md-* class to the opening tag, where * is a number between 1 and 12.
  • Example: <div class="col-md-6">

Putting it all together:

						
							<section class="row">
								<div class="col-md-6"></div>
								<div class="col-md-6"></div>
							</section>
