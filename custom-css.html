This page focuses on your style.css document.

Bootstrap provides CSS that you can add to your site by incorporating built-in classes

For example, the class="col-md-*" that you add to the columns in your grid.

If you want to incorporate custom styles, like background colors, font colors, etc, add those to your style.css document.

You do not need to add @import 'reset.css'; to your style.css document.

This is because Bootstrap has a reset built into it.

You may not be able to reuse the exact same styles that you have created for previous assignments.

This is because Bootstrap uses its own classes to style things.

Example: adding a background color to your page

In your previous assignment, you added #page to your style.css document.

Bootstrap uses class="container" to define the boundaries of your webpage in HTML.

Use .container in your style.css document to apply styles to your webpage.

Example: adding a custom font color

You can target text elements in your style.css document and apply colors to them.

If you want only one item on your page to be a different color - for example, a specific <h1>, you can apply an id to that item.

For example: <h1 id="purple"> in the HTML document, and in style.css the following is added:

					#purple {
						color: color-code-here;
					}

To give the final result:

A purple heading.
